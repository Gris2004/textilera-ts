# Laboratory Practice
a textile manufacturer's website that adds new fabrics received in the packaging department
# Dependencies
- typescript
    - typescript
    - ts-node
- @types/node
- @types/express
- express
- nodemon
- jsonwebtoken
- mongoose

# Features
- show all fabrics
- show fabric by id and name
- edit a record by id
- delete record with a confirmation message

# Fabric Model
- fabric id: number
- fabric name: string
- fabric color: string
- fabric quality: string
- fabric entry date: string

# DataBase
use mongodb
client: mongosh and API
